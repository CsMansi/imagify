# Imagify — Text-to-Image Generator

Imagify is a full-stack web application that allows users to convert text into AI-generated images using their creativity and imagination.

## 🔑 Features

- 🔐 **Google Authentication** – Login securely using your Gmail account.
- 🎨 **Text-to-Image Generator** – Converts simple text prompts into AI-generated images.
- 🎟️ **5 Free Credits** – New users get 5 free image generations.
- 💳 **Razorpay Payment Gateway** – Buy additional credits after using free credits.
- 👤 **User Dashboard** – View and manage your remaining credits.
- 📊 **Purchase History** – Track your past credit purchases.

## 🛠️ Technologies Used

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** Google OAuth 2.0
- **Payments:** Razorpay Payment Gateway
- **Image Generation API:** Stability AI API / OpenAI DALL·E (Edit based on your project)
